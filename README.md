# a2agora.github.io

Source for the **[a2agora.org](https://a2agora.org)** website — the landing page
and documentation hub for **A2Agora / ACMP** (Agent Compute Market Protocol).

Built with [Jekyll](https://jekyllrb.com) + [just-the-docs](https://just-the-docs.com),
served by GitHub Pages.

- Protocol specification: [github.com/a2agora/spec](https://github.com/a2agora/spec)
- Reference SDK: [github.com/a2agora/sdk-reference](https://github.com/a2agora/sdk-reference)

## Local preview

```bash
bundle install
bundle exec jekyll serve
```

Machine-readable entry points: [`/llms.txt`](llms.txt), `/sitemap.xml`, `/robots.txt`.
